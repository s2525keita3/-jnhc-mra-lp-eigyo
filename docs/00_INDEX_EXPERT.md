## 📌 運用ルール（整理の前提）

- **表示・ビルドには一切影響しない**: ドキュメントの配置・名前変更はアプリの動作を変えません。
- **00〜10**: 開発・運用の「公式」参照。番号順で依存関係を意識して読めます。
- **strategy / proposals**: 戦略・提案の参照用。必要時のみ参照します。
- **ルート**: README と設定ファイルのみにし、見やすさを保ちます。
